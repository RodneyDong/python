# Markdown Tricks

❓ How do I create a table?
✔️ Use the | | symbol 
ex:
|operator | LaTex Symbols |
|---------|---------------|
× | times
÷ | div
± | pm
❓ How do I enter section title?
✔️ Use ## for big titles and ### for subtitles

❓ How do I make a table of content
✔️ right click - command palette - then search for TOC
